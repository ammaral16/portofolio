Tentang Saya

Halo! Saya Saya Ammar Al Hakim Asshiddiqi, seorang UI/UX Designer fresh graduate dengan semangat tinggi dan dedikasi untuk menciptakan pengalaman pengguna yang menarik dan intuitif. Lulusan dari Universitas Ahmad Dahlan di bidang Informatika, saya telah mendapatkan pemahaman yang kuat tentang prinsip-prinsip desain, prototyping, dan user testing. Selama masa studi, saya aktif dalam berbagai proyek desain yang melibatkan pembuatan user interface untuk aplikasi mobile dan website. Saya selalu tertarik dengan bagaimana desain dapat mempengaruhi interaksi pengguna dan berusaha untuk membuat solusi yang tidak hanya estetis, tetapi juga fungsional. Saya percaya bahwa desain yang baik adalah hasil dari proses iteratif, di mana umpan balik pengguna menjadi bahan bakar untuk perbaikan yang terus-menerus. Dengan keterampilan dalam tools seperti Figma, Adobe XD, dan Sketch, serta pemahaman tentang HTML dan CSS dasar, saya siap untuk menghadapi tantangan dunia kerja. Saya sangat antusias untuk berkontribusi dalam tim yang dinamis, di mana saya bisa terus belajar dan mengembangkan keterampilan saya lebih lanjut. Jika Anda mencari seorang desainer yang kreatif, proaktif, dan berorientasi pada detail, saya akan dengan senang hati menjadi bagian dari proyek Anda berikutnya.
